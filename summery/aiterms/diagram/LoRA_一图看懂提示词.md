# LoRA 一图看懂图片生成提示词

## 生成目标

生成一张横版 16:9 的中文手绘教学卡片，用于 AI 词条详情页正文开头，解释 LoRA 如何在不改动基础模型主体的情况下，通过训练少量低秩适配器来适配新任务。画面应是中密度资料小总结，清晰、克制、适合长期知识库网站。

## 画面构图

主构图使用分层型。画面中央是一台大型基础模型机器或堆叠模型层，主体上有小锁头，表示“原始权重冻结”。模型右侧或旁边插入两块可拆卸小模块，标记为 LoRA 适配器和小矩阵。左侧有“新任务 / 新风格”的输入卡片，箭头进入基础模型和适配器组合；右侧输出“适配后回答 / 风格”的结果卡片。右下角放一个黄色边界提醒便签。

不要做成横向 4 步流程图，不要做成中心辐射图。主视觉必须突出“冻结主体 + 旁路适配器”的结构关系。

## 固定风格

16:9 横版中文手绘教学卡片。米白色纸张背景，马克笔 / 签字笔线条，轻微手绘感但清晰工整。使用细线圆角边框和柔和色块：浅绿、浅紫、浅黄、浅蓝、浅灰。可用小图标、锁头、插片、模型层、便签、箭头和小标签。整体可爱但克制，不要真实照片、复杂 3D、赛博朋克、霓虹、营销海报感、伪造品牌 Logo 或假 UI 截图。

## 画面元素

- 中央：大型“基础模型”机器或堆叠层，主体用浅蓝灰色，带锁头。
- 旁路：两块小型可拆卸适配模块，用浅绿和浅紫色，插在模型旁边。
- 左侧：输入卡片表示新任务或新风格。
- 右侧：输出卡片表示适配后的回答或风格。
- 右下角：黄色便签作为边界提醒。
- 小箭头：从新任务到模型，再到输出；适配器和模型之间用短线连接。

## 图中文字

只使用 brief 中“最终入图文字清单”列出的文字。

请逐字保留以下简体中文文字，不要改写、增补或生成伪文字：

- LoRA：不改主体的小微调
- 基础模型
- 原始权重冻结
- LoRA 适配器
- 只训练小矩阵
- 适配新任务
- 可切换保存
- 主体模型不重训
- 学习少量变化量
- 显存压力更低
- 仍要好数据评估
- 不是独立小模型
- LoRA 的关键是少改参数，让大模型学会新用法。

所有主要文字必须清晰可读，不要太小，不要出框，不要乱码，不要伪文字。英文术语 LoRA 可以保留，但其他说明使用简体中文。

## 准确性与禁忌

不要编造公司名、产品名、数据、发布日期、性能比例或社区共识。不要暗示 LoRA 可以脱离基础模型独立运行。不要写“免费微调”“完全替代全量微调”“一定接近全量微调”“自动保证安全”。不要绘制未经确认的官方 Logo。不要把 QLoRA、DoRA、AdaLoRA 等变体放入主图。

## 输出规格

输出横版 16:9 位图。优先生成清晰可读的 PNG、JPG、JPEG 或 WebP。整体信息密度为中密度：能看懂 LoRA 的主要结构、价值和边界，但不能像资料报告一样满屏小字。重要文字和图形不要贴边，保留适合网页正文开头展示的安全边距。

## 失败重试提示

如果生成图中文字出现乱码、错别字、伪文字或不可读，请保留同一分层构图，但减少辅助说明，只保留标题、6 个短标签、1 个边界提醒和底部总结后重试一次。不要用程序覆盖修字，不要改用 SVG、HTML、PPT 或代码绘图兜底。
